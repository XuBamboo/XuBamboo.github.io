Server Introduction

Chinese name: 北冥鱼:我的世界基岩版玩家联盟. English name: Minecraft: Bedrock Edition Players' Union, referred to as MCBEPU.MCBEPU Network is an alliance based on multiple non-profit Bedrock Edition servers, aiming to provide players with more choices. Any player on any server of Beimingyu is equal, there is no paid service, and all servers are public service and follow the EULA agreement. In addition, the open gameplay allows players from different servers to explore the true charm of this game with like-minded friends. We hope to create a good game environment and prohibit anyone from abusing and attacking other players. And we hope that all players can have a happy game time, so please remember to keep a good mood when entering the server and communicate with other players more.
