Description

Minecraft:Bedrock Edition Players' Union,referred to as MCBEPU。 This alliance covers multiple modes such as single-player and multiplayer. At present, the unified name of the server-based massively multiplayer online is MCBEPU Network. MCBEPU Network is an alliance based on a number of non-profit Bedrock Edition servers, aiming to provide players with more options for gameplay. Any player on any server of MCBEPU is equal, there is no fee-based service, and all servers are public service and follow the EULA agreement. In addition, the open gameplay allows players on different servers to explore the true charm of this with like-minded friends. We hope to create a good gaming environment and prohibit anyone from insulting or attacking other players. And we hope that all players can spend happy game time happily, so please remember to keep a good mood when entering the server and communicate with other players more.
